类

在介绍元类之前，我们先来聊一下类。在大多数语言中，类是用于生成对象的一段封装的代码，Python也是如此

1
2
3
4

class ObjectCreator(object):
    pass
my_object = ObjectCreator()
print(my_object) # <__main__.ObjectCreator at 0x7f4b78437550>

ObjectCreator类生成了my_object对象，但在Python中，一切皆对象，即ObjectCreator也是对象

一旦使用了关键字class，Python便会在内存中生成一个名为”ObjectCreator”的对象

1
2

    pass

ObjectCreator是一个类，所以它能够生成实例对像。而类也是对象，所以我们能够对ObejctCreator执行以下操作：

  • 把它赋给一个变量
  • 复制这个对象
  • 往对象里添加属性
  • 作为参数传递给函数

例如：

使用type动态创建类

因为类是对象，所以我们可以像创建对象那样动态创建类

    else:

其实上面的代码算不上动态，因为我们还是自己用class定义了类。我们知道类也是对象，那么类必然也是由”类的类”生成的。
而这个”类的类”其实就是我们这篇文章的主题-元类。 当我们使用class关键字定义类的时候，Python会自动创建这个类。当然，Python也给我们提供了手工创建类的手段。
还记得type()吗？我们可以调用这个函数知道查看对象的类型

除了查看对象的类型，type()还有另外一个功能，它能够动态创建类。如果我们想用type()创建类，我们需要传入3个参数，

元类

前面已经看到，type()可以用于生成类，就像下面这样

1

MyClass = type('MyClass', (), {})

而type()之所以可以创建类，因为它事实上是一个元类。type()是Python用于生成所有类的元类。 Python一切皆对象，整形是对象，字符串是对象，函数是对象，类也是对象。我们可以通过__class__ 属性查看其对应的类

1
2
3
4
5
6

9

age = 35
print(age.__class__) # <class 'int'>
name = 'bob'
print(name.__class__) # <class 'str'>
def foo(): pass
print(foo.__class__) # <class 'function'>
class Bar(object): pass
b = Bar()
print(b.__class__) # <class '__main__.Bar'>

那么，class的class会是什么呢

1
2
3
4

print(age.__class__.__class__) # <class 'type'>
print(name.__class__.__class__) # <class 'type'>
print(foo.__class__.__class__) # <class 'type'>
print(b.__class__.__class__) # <class 'type'>

我们可以看到int、str、function对应的类都是type类

定制元类

元类用于生成类对象，我们可以称其为类工厂。type是Python内置的元类，当然我们也可以定制自己的元类
使用元类的目的在于创建类时，可以动态地改变类
假设有一个这样的需求，我们想把类的属性都定义为大写那么，我们只要告诉元类，要把类的所有属性都转为大写即可 元类可以是任意的可调用对象，不必须是一个类 例如：

class UpperAttrMetaclass(type):
    
    def __new__(cls, clsname, bases, dct):
        # 挑选不以__开头的属性并把其转为大写
        uppercase_attr = {}
        for name, val in dct.items():
            if not name.startswith('__'):
                uppercase_attr[name.upper()] = val
            else:
                uppercase_attr[name] = val
        # 使用type创建类
        return super(UpperAttrMetaclass, cls).__new__(cls, clsname, bases, uppercase_attr)
1
2
3

 class Foo(metaclass=UpperAttrMetaclass):

    bar = 'bip'
1
2

print(hasattr(Foo, 'bar')) # False
print(hasattr(Foo, 'BAR')) # True

# ORM: object relational mapping 对象-关系映射
# 把关系数据库的一行映射为一个对象，也就是一个类对应一个表

class Field:
    
    def __init__(self, name, column_type): # column_type: 列类型
        self.name = name
        self.column_type = column_type
        
    def __str__(self):
        return '{}:{}'.format(self.__class__.__name__, self.name)
    

class IntegerField(Field):
    
    def __init__(self, name):
        super().__init__(name, 'bigint')
        
        
class StringField(Field):
    
    def __init__(self, name):
        super().__init__(name, 'varchar(100)')
        
        
class ModelMetaclass(type):
    
    def __new__(cls, name, bases, attrs):
        if name == 'Model': # 如果是Model基类，则直接返回
            return super().__new__(cls, name, bases, attrs)
        print('Found model: {}'.format(name))
        mappings = dict()
        for k, v in attrs.items():
            if isinstance(v, Field):
                print('Found mappings:{}==>{}'.format(k, v))
                mappings[k] = v
        for k in mappings.keys():
            attrs.pop(k)
        attrs['__mappings__'] = mappings
        attrs['__table__'] = name # 表名
        return super().__new__(cls, name, bases, attrs)
    
    
class Model(dict, metaclass=ModelMetaclass):
    def __init__(self, **kw):
        super().__init__(**kw)
    
    # 既可以通过d[k]，也可以通过d.k获取值
    def __getattr__(self, key):
        try:
            return self[key]
        except KeyError:
            raise AttributeError('Model object has no attrubute {}'.format(key))
    
    # 允许动态设置key的值，不仅可以d[k]，还可以d.k
    def __setattr__(self, key, value):
        self[key] = value
        
    def save(self):
        fields = []
        params = []
        args = []
        for k, v in self.__mappings__.items():
            fields.append(v.name)
            params.append('?')
            args.append(getattr(self, k, None))
        sql = 'insert into {} ({}) values ({})'.format(self.__table__, ','.join(fields), '.'.join(params))
        print('SQL:{}'.format(sql))
        print('ARGS:{}'.format(args))
        
        
class User(Model):
    
    id = IntegerField('id')
    name = StringField('username')
    email = StringField('email')
    password = StringField('password')

class SingletonMeta(type):
    _instances = {}
    def __call__(cls, *args, **kwargs):
        if cls not in cls._instances:
            cls._instances[cls] = super().__call__(*args, **kwargs)
        return cls._instances[cls]


class SingletonClass(metaclass=SingletonMeta):
    pass
