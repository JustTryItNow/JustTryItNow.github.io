堆是一种特殊的二叉树结构，根据根节点与左右节点的值分最小堆和最大堆。最小堆中每个根节点的值总小于或者等于左右节点的值，最大堆中每个根节点的值总大于或者等于左右节点的值。heapq是最小堆。

heapq–最小堆

import heapq
heap = [4, 3, 1, 2, 5]
heapq.heapify(heap)
# 访问最小元素，不会弹出
print("最小元素: ", heap[0])  # 最小元素: 1
while heap:
    heapq.heappop(heap)  # heappop()弹出最小元素
    # 1
    # 2
    # 3
    # 4
    # 5

# 也可以通过压入元素的方式创建堆队列
heap1 = []
for i in [4, 3, 1, 2, 5]:
    heapq.heappush(heap1, i)
# 访问最小元素，不会弹出
print("最小元素: ", heap1[0])  # 最小元素: 1
while heap1:
    heapq.heappop(heap1)  # heappop()弹出最小元素
    # 1
    # 2
    # 3
    # 4
    # 5

nlargest(n, iterable, key=None)

从iterable所定义的数据中返回前n个最大的元素组成的列表，如果提供了key则其应指定一个单参数的函数，用于从iterable的每个元素中提取比较键。n较大时，使用sorted()更好

1
2
3
4
5
6

import heapq
print(heapq.nlargest(2, [3, 1, 5, 7, 2, 4]))  # [7, 5]

data = [{"name": "hello", "age": 10}, {"name": "hi", "age": 15}, {"name": "h", "age": 5}]
print(heapq.nlargest(2, data, key=lambda item: item["age"]))
# [{'name': 'hi', 'age': 15}, {'name': 'hello', 'age': 10}]

nsmallest(n, iterable, key=None)

从iterable所定义的数据中返回前n个最小的元素组成的列表，如果提供了key则其应指定一个单参数的函数，用于从iterable的每个元素中提取比较键。n较大时，使用sorted()更好

import heapq
print(heapq.nsmallest(2, [3, 1, 5, 7, 2, 4]))  # [1, 2]

data = [{"name": "hello", "age": 10}, {"name": "hi", "age": 15}, {"name": "h", "age": 5}]
print(heapq.nsmallest(2, data, key=lambda item: item["age"]))
# [{"name": "h", "age": 5}, {'name': 'hello', 'age': 10}]
